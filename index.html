pulse

pulse is a small node.js server and client that pipes MIDI Clock messages to your browser through a WebSocket. It also interpolates the bpm and current phase. Simply run the server and use: 

pulse.connect('http://yourserver:9000');
Pulse supplies a number of ways to use this information:
pulse.bpm
The currently detected beats per minute.
pulse.beat()
This function returns the current beat pluse phase: e.g. 12.4 means that we're about halfway through the thirteenth beat. Use a modulo of 1 to get only the phase.
pulse.pulse()
This convenient function supplies a pulsing value between 0 and 1 that reaches a peak on every whole beat, and decays after that.

You want to see it in action, don't you? Well, if you are running the server, connect to it below. Oh, you're not? You can just tap a beat to see the time tracking in action. Check out the source code to this page to see how the data provided by pulse is used to animate the page. Everything is on GitHub.
